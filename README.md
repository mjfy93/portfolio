
# Portfolio

A personal portfolio website built with React Router v7 and Vite, featuring custom icon navigation derived from static imagery.

## Tech Stack

- React 19
- React Router v7 (Framework Mode)
- Vite
- Font Awesome
- CSS3

## Features

- File-based routing with React Router v7
- Theme context for light/dark mode support
- Custom icons created from royalty-free background imagery
- Quick-access links to GitHub and Vercel profiles
- Responsive navigation component

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/portfolio.git

# Navigate to project directory
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```
